# BackEnd

## prerequisites
- nodejs
- mongodb
- elasticsearch

    Install ![ElastSearch](https://www.elastic.co/downloads/elasticsearch) and export bin folder, e.g.

    ```bash
    echo 'export PATH=/Users/ytian/Applications/elasticsearch-1.5.2/bin:$PATH' >> ~/.bash_profile; source ~/.bash_profile
    ```

## set up
```shellscript
npm install
npm start
```
